# Garimpo Secreto

Aplicação web desenvolvida com **Next.js e TypeScript** para explorar e compartilhar descobertas digitais.

## 🚀 Demonstração

A aplicação está disponível online:

https://garimpo-secreto-tf5a.vercel.app/

## 🛠️ Tecnologias utilizadas

- Next.js
- React
- TypeScript
- Node.js
- Vercel

## 📂 Estrutura do projeto

app/ → páginas e rotas da aplicação  
lib/ → funções auxiliares e lógica da aplicação  
public/ → arquivos estáticos  

## ⚙️ Como executar o projeto localmente

Clone o repositório:

git clone https://github.com/phferolla/garimpo-secreto

Entre na pasta do projeto:

cd garimpo-secreto

Instale as dependências:

npm install

Execute o projeto:

npm run dev

A aplicação estará disponível em:

http://localhost:3000

## 📌 Objetivo do projeto

Este projeto foi desenvolvido para praticar desenvolvimento web moderno utilizando **Next.js**, organização de projeto e deploy em produção.

## 🌐 Deploy
